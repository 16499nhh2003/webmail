- Cách setup và chạy project:
  1. Download hoặc clone project về từ GitLab
  2. Mở XAMPP -> Truy cập PHPMyAdmin và import file "database.sql" để khởi tạo cơ sở dữ liệu cùng một số dữ liệu mẫu
  3. Copy toàn bộ nội dung trong thư mục "source" và paste vào thư mục "htdocs" trong XAMPP
  4. Mở file "db.php" để cấu hình lại database (hostname, username, password) nếu có thay đổi tùy theo máy. Mặc định trên máy
      tính này là: hostname: 127.0.0.1, username: root, password: ''
  5. Mở trình duyệt và truy cập vào localhost để chạy project
